# Crowd-Detection-Using-CSRnet
This Repo is for Developing a python software to estimate the count of people in a crowd and finding crowd density.
#Datasets
ShanghaiTech Dataset: Google Drive
#Results
Shanghai A MAE: 39.2

Shanghai B MAE: 8.32
